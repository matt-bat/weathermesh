# Changelog

Project credit: Matthew Bateman.

## 0.1.0

Initial release candidate.

- Added point forecast aggregation from NOAA/NWS and Open-Meteo.
- Added locality guardrails and source-vs-average comparisons.
- Added hourly, daily, weekly, and 14-day forecast views.
- Added a Three.js globe with country, region, and locality layers.
- Added Natural Earth country-boundary sampling for globe averages.
- Added provider timeouts, retries, cache bounds, and validation.
- Added browser verification for globe rendering.
