# Attribution

Project credit: Matthew Bateman.

Aggregate Weather uses a few public/open data sources and libraries. This page is the human-readable version of what the app also exposes at `/api/attribution`.

## Weather Data

- NOAA/National Weather Service API  
  Used for U.S. point forecasts. NWS describes the API as open government weather data and asks applications to send a clear User-Agent.

- Open-Meteo  
  Used for model forecast data and globe map sampling. The free API is intended for non-commercial use and Open-Meteo data is provided under its published terms.

## Map Data

- Natural Earth  
  Used for country boundaries. Natural Earth map data is public domain.

## Rendering

- Three.js  
  Used for the interactive WebGL globe.

## Notes

If this app is used commercially, review Open-Meteo's current commercial-use terms before launch.
